# Android Memory Game

An Android memory game developed as part of the **Mobile App Development module at ATU Sligo**.

The game displays an increasingly long sequence of coloured circles for the player to memorise. The player repeats the sequence by tilting the device in the corresponding directions. Scores are stored locally, allowing players to view the top five high scores.

## Features

- Randomly generated memory sequences
- Accelerometer-based player input
- Increasing difficulty after each successful round
- Local high-score storage using SQLite
- Landscape-oriented interface

## Built With

- Java
- Android SDK
- XML layouts
- SQLite

## Running the App

Open the project in Android Studio, allow Gradle to sync, and run it on an Android device or emulator with API level 29 or later. A physical device is recommended for the accelerometer controls.
